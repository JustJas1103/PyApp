# AI-Powered Filipino Menu Recommender

## Project Structure

```bash
├── README.md
├── recipe.json
└── src
    ├── __init__.py
    ├── main.py
    └── recommender.py
```

## Description

This project provides a menu recommender system that suggests Filipino recipes based on available ingredients using an AI model.

## Getting Started

These instructions will help you set up the project locally.

### Prerequisites

- Python 3.8 or higher
- Pip

### Installing

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/filipino-menu-recommender.git
   ```
2. Change directory to the project folder
   ```bash
   cd filipino-menu-recommender
   ```
3. Install the required packages
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To use the menu recommender, run the following command:

```bash
python src/main.py
```
